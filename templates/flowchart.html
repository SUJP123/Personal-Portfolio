Flowchart Generator

Visual representation and app to create flow charts.

Overview

This project is a Java-based Flowchart UI application built using the Swing GUI library. The application allows users to create, connect, and manipulate tasks represented as boxes in a flowchart. Users can establish connections between tasks, dragging connectors with control points to visualize flow between tasks. The UI supports dynamic resizing of task boxes based on label length and introduces customizable shapes for tasks, providing a versatile and interactive flowcharting experience.

Key Features

  • Task Creation and Manipulation: Create resizable tasks with customizable shapes and labels.
  • Connection Establishment: Establish connections between tasks using arrows with adjustable control points.
  • Dynamic Textbox Sizing: Task box sizes adapt to label lengths for an aesthetically pleasing layout.
  • Customizable Shapes: Choose from various shapes (e.g., rectangles, ellipses) for individual tasks.
