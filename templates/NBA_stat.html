NBA Stats Predictor

Predict NBA stats, season leaders, and MVP winners.

Overview

This project integrates web scraping and machine learning to analyze and predict NBA player statistics. The data is gathered by systematically scraping regular season and playoff stats for each player since the 2014-15 season from the NBA website. The web scraping process incorporates random delays to mimic human behavior and avoid detection. Using a Random Forest Regressor, the model is trained to predict player statistics for key features. Predictions are then applied to individual players, showcasing expected performance for the upcoming season. Additionally, the project includes a feature to identify potential MVP candidates based on weighted scores derived from predicted statistics.

Process and Key Features

  • Web Scraping Infrastructure:
    • URL Dependencies Analysis: Examining the URL structure to understand dependencies for web scraping.
    • Dynamic Data Retrieval: Using the Requests library to fetch JSON data from the NBA website's API.
    • Data Structuring: Utilizing Beautiful Soup for parsing HTML and structuring data into Pandas DataFrames.
  • Data Collection and Processing:
    • Seasonal Iterations: Systematically scraping player statistics for both regular seasons and playoffs since the 2014-15 NBA season.
    • Randomized Delays: Incorporating random delays with NumPy to simulate human-like behavior during web scraping.
    • Data Concatenation: Merging individual player statistics with additional information like the season and type (regular season or playoffs).
  • Machine Learning Model:
    • Random Forest Regressor: Implementing Scikit-learn's Random Forest Regressor for predicting future player statistics.
    • Feature Selection: Focusing on key player performance features such as points, assists, rebounds, steals, and blocks.
    • Individual Player Predictions: Applying the trained model to predict statistics for specific players in the upcoming season.
  • Evaluation and Visualization:
    • Model Evaluation Metrics: Assessing model performance using metrics like Mean Squared Error (MSE) and Mean Absolute Error (MAE).
    • Scatter Plots and Residuals: Visualizing actual vs. predicted values and model residuals using Matplotlib.
    • Predicted Stats Display: Displaying predicted statistics for specific players, providing insights into their potential future performance.
  • MVP Candidate Identification:
    • Weighted Score Calculation: Determining MVP candidates by calculating weighted scores based on predicted statistics.
    • Top Performer Identification: Identifying top players with the highest MVP scores for consideration.
  • Development Environment:
    • Python Language: Employing Python as the primary programming language for the entire project.
    • Library Integration: Leveraging key libraries like Pandas, NumPy, Scikit-learn, and Matplotlib for data manipulation, analysis, and visualization.
