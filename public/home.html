Facial Recognition Model

This MATLAB project involves fine-tuning a SqueezeNet model for face recognition using a dataset of facial images. The code includes data preparation, model configuration, training, and testing phases, ultimately evaluating the trained model's accuracy on a validation set and visually assessing its performance on randomly selected images. The goal is to create a face recognition system capable of distinguishing between different individuals based on their facial features.

Technologies Used

  • MATLAB: The main programming environment for implementing and executing the entire project.

  • SqueezeNet Model: A pre-trained convolutional neural network architecture employed as the base model for face recognition.

Current Research

I'm currently working as a research assistsant at Nazockdast Labs under the Applied Sciences and Engineering Department at UNC Chapel Hill. In this lab we use Python, C++, and MATLAB to study particle and crowd dynamics, and to create simulations that accurately model particle physics. Currently, I'm collaborating with a few other undergraduate students in order to model a group of particles going through a gate and accurately simulating the brownian motion of the particles.

In order to create these simulations, we utilize the HOOMD-blue package in python. HOOMD-blue provides a high-level interface for setting up particle simulations, and it is designed to efficiently leverage both CPUs and GPUs for accelerated performance. Users can specify the interactions between particles, set simulation parameters, and run molecular dynamics simulations to observe the behavior of particle systems over time. I will keep this section updated as our projects progress.
